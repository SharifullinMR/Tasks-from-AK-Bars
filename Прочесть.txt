Пусть в нашем датасете клиенты - clients.csv есть следующие столбцы:

id_client - int - Уникальный идентификатор клиента
full_name - string -  ФИО клиента.
active_flag - bool - Признак  активности клиента(активный/неактивный)   
balance - float -  Остаток денег на счетах клиента
country - string - Страна проживания клиента
salary - float - Зарплата клиента
gender - bool - Пол клиента: 1 — мужчина, 0 — женщина.  
rating - int -  Кредитный рейтинг клиента
card_type - string - Тип банковской карты
leave_client - bool - Ушёл ли клиент из банка
